011    /**
012     * Utility class for retrieving JSON-RPC 2.0 positional parameters (packed into
013     * a JSON Array). 
014     *
015     * <p>Provides a set of getter methods according to the expected parameter type
016     * (number, string, etc.) and whether the parameter is mandatory or optional:
017     *
018     * <ul>
019     *     <li>{@code getXXX(param_pos)} for mandatory parameters, where {@code XXX}
020     *         is the expected parameter type.</li>
021     *     <li>{@code getOptXXX(param_pos, default_value)} for optional parameters,
022     *         specifying a default value.</li>
023     * </ul>
024     *
025     * <p>There are also generic getter methods that let you do the type conversion 
026     * yourself.
027     *
028     * <p>If a parameter cannot be retrieved, e.g. due to a missing mandatory 
029     * parameter or bad type, a 
030     * {@link com.thetransactioncompany.jsonrpc2.JSONRPC2Error#INVALID_PARAMS}
031     * exception is thrown.
032     *
033     * <p>Example: suppose you have a method with 3 positional parameters where the
034     * first two are mandatory and the last is optional and has a default value of
035     * {@code true}.
036     *
037     * <pre>
038     * // Parse received request string
039     * JSONRPC2Request request = null;
040     *
041     * try {
042     *         request = JSONRPC2Request.parse(jsonString);
043     * } catch (JSONRPC2ParseException e) {
044     *         // handle exception...
045     * }
046     *
047     * // Create a new retriever for positional parameters
048     * List params = (List)request.getParams();
049     * PositionalParamsRetriever r = new PositionalParamsRetriever(params);
050     *
051     * try {
052     *         // Extract first mandatory string parameter
053     *         String param1 = r.getString(0);
054     *
055     *         // Extract second integer parameter
056     *         int param2 = r.getInt(1);
057     *
058     *         // Extract third optional boolean parameter which defaults to true
059     *         boolean param3 = r.getOptBoolean(2, true);
060     *
061     * } catch (JSONRPC2Error e) {
062     *         // A JSONRPC2Error.INVALID_PARAMS will be thrown to indicate
063     *         // an unexpected parameter type or a missing mandatory parameter.
064     *         // You can use it straight away to create the appropriate
065     *         // JSON-RPC 2.0 error response.
066     *         JSONRPC2Response response = new JSONRPC2Response(e, null);
067     * }
068     * </pre>
069     *
070     * @author Vladimir Dzhuvinov
071     * @version 1.25.1 (2011-10-08)
072     */
073    public class PositionalParamsRetriever

145             * Throws a {@code JSONRPC2Error.INVALID_PARAMS} exception if there is
146             * no parameter at the specified position, its value is {@code null}, 
147             * or its type doesn't map to the specified.
148             *
149             * <p>You may use this method to fire the proper JSON-RPC 2.0 error
150             * on a missing or badly-typed mandatory parameter.
151             *
152             * @param position The parameter position.
153             * @param clazz    The corresponding Java class that the parameter 
154             *                 should map to (any one of the return types of the 
155             *                 {@code getXXX()} getter methods. Set to 
156             *                 {@code Object.class} to allow any type.

162                    throws JSONRPC2Error {
163

168            /**
169             * Throws a {@code JSONRPC2Error.INVALID_PARAMS} exception if there is
170             * no parameter at the specified position or its type doesn't map to the
171             * specified.
172             *
173             * <p>You may use this method to fire the proper JSON-RPC 2.0 error
174             * on a missing or badly-typed mandatory parameter.
175             *
176             * @param position  The parameter position.
177             * @param clazz     The corresponding Java class that the parameter 
178             *                  should map to (any one of the return types of the 
179             *                  {@code getXXX()} getter methods. Set to 
180             *                  {@code Object.class} to allow any type.
181             * @param allowNull If {@code true} allows a {@code null} parameter
182             *                  value.
183             *
184             * @throws JSONRPC2Error On a missing parameter or bad type 
185             *                       ({@link JSONRPC2Error#INVALID_PARAMS}).
186             */
187            public <T> void ensureParameter(final int position, final Class<T> clazz, final boolean allowNull)
188                    throws JSONRPC2Error {
189                    
190                    // First, check existence only
191                    ensureParameter(position);
192                    
193                    // Now check type
194                    Object value = params.get(position);
195                    
196                    if (value == null) {
197                            
198                            if (allowNull)
199                                    return; // ok
200                            else
201                                    throw JSONRPC2Error.INVALID_PARAMS;
202                    }
203                    
204                    if (! clazz.isAssignableFrom(value.getClass()))
205                            throw JSONRPC2Error.INVALID_PARAMS;
206            }
